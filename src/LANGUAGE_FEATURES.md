# Blitz App - 多语言功能

## 支持的语言

Blitz应用现在支持8种语言：

1. 🇺🇸 **English** (默认)
2. 🇨🇳 **简体中文**
3. 🇹🇼 **繁體中文**
4. 🇪🇸 **Español** (西班牙语)
5. 🇳🇱 **Nederlands** (荷兰语)
6. 🇮🇹 **Italiano** (意大利语)
7. 🇫🇷 **Français** (法语)
8. 🇩🇪 **Deutsch** (德语)

## 如何切换语言

在主页（Home）顶部，您会看到一个带有国旗图标和地球仪图标的按钮。点击它可以打开语言选择菜单，选择您想要的语言即可立即切换。

## 特点

- ✅ **无缝切换**：语言切换即时生效，无需刷新页面
- ✅ **完整翻译**：所有页面和功能都已完整翻译
- ✅ **持久化**：语言选择会在整个应用中保持一致
- ✅ **视觉标识**：每种语言都有对应的国旗图标，便于识别

## 技术实现

- React Context API 用于全局语言状态管理
- 完整的翻译文件系统，包含所有文本内容
- 组件级别的翻译集成，确保所有文本都能正确翻译

## 翻译覆盖范围

所有7个主要页面的文本都已翻译：
1. Main Page (主页) - 自行车信息、地图导航
2. Statistics (统计) - 所有统计数据和图表
3. Activities (活动) - 城市打卡、碳足迹、能量兑换、驿站补给
4. Community (社区) - 社交功能
5. Me (我的) - 个人资料和设置
