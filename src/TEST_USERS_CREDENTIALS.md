## 数据库插入说明

### 方法 1: 使用 Supabase Dashboard
1. 在 Supabase Dashboard 中，进入 Authentication > Users
2. 手动创建三个用户账户（使用上述邮箱和密码）
3. 记录每个用户的 UUID
4. 在 SQL Editor 中运行 `/scripts/insert_test_users.sql`
5. 将 SQL 中的 `USER_ID_1`, `USER_ID_2`, `USER_ID_3` 替换为实际的 UUID

### 方法 2: 使用 TypeScript 脚本
1. 获取 Supabase Service Role Key（在 Project Settings > API）
2. 在 `/scripts/insertTestUsers.ts` 中替换 `YOUR_SUPABASE_SERVICE_ROLE_KEY`
3. 运行脚本：`npx tsx scripts/insertTestUsers.ts`

### 方法 3: 使用 Supabase CLI
```bash
# 安装 Supabase CLI
npm install -g supabase

# 登录
supabase login

# 链接到项目
supabase link --project-ref utvozryyrirwllumubqn

# 运行 SQL 脚本
supabase db execute -f scripts/insert_test_users.sql
```

## 注意事项

1. **密码安全性**: 这些密码仅用于测试环境，请勿在生产环境中使用类似密码
2. **数据一致性**: 确保在插入数据前，数据库表结构已正确创建
3. **时区设置**: 所有时间戳使用 UTC 时区
4. **语言切换**: 用户可以在应用内切换语言，但默认语言基于账户设置
